# Гайд

## Mentor Passives

1. **Chi Dash** — Активация: C+C. КД: 3 сек. Быстрый рывок вперёд, хитбокс которого ломает обычный/улучшенный блок.
   - Требуется 6400 чи и 6400 нин

2. **Instant Rush / Faster STM Regen** — Раш (W+W) теперь мгновенный, в нём можно использовать скиллы, но он всё ещё не пробивает улучшенный блок. Стамина восстанавливается значительно быстрее. Скип хилки работает мгновенно.
   - Требуется 7100 тай

3. **Stronger Block** — Большинство блокбрейкеров не ломают ваш блок. Остальные ломают значительно дольше. Блок восстанавливается быстрее. Трата: 100+ STM / 0.6 сек.
   - Требуется 7000 хп

4. **Wider Dash** — Деш становится длиннее. Стакается с модами, использующими instant dash (например, Shock Cloak). Трата: 600 STM / деш.
   - Требуется 6200 тай

5. **Faster / Passive Charging** — Чакра восстанавливается в 1.5–2 раза быстрее, плюс пассивный реген 2–3к в секунду.
   - Требуется 6500 нин

### Jiso Seishin (R1/C3)

- Подходит для игроков, предпочитающих защиту.
- Strong Block, быстрая регенерация блока, высокое хп и быстрая чакра делают вас лучшим танком в игре.
- Самый простой в прокачке.
- Хорош в массовых PvP: рейды, вары и т.д.
- За счёт увеличенного хп хилка регенит больше обычного.

**Хорош с расами:** Celestial, Puppet

### ✮ Shindai Akuma (R2/C2)

- Самый мобильный ментор в игре. Chi Dash + Wider Dash делают вас крайне быстрыми.
- Комбинируйте TP-мув с Chi Dash — мгновенно ломаете улучшенный блок оппонента.
- Вместо улучшенного блока используйте чакра-прыжки (Блок + Прыжок).
- Сбалансированные статы между тай и чи скиллами. Подходит для любого стиля игры.

**Хорош с расами:** Puppet, Celestial

### Narumaki (R3/C4)

- Больше чакры — больше дамага.
- Подходит **только** для командных боёв: 2v2, вары и т.д.

::: danger Не качай в соло
Narumaki бесполезен в 1v1 — только рейды и вары.
:::

**Хорош с расой:** Celestial

### Bruce Kenichi (R4/C1)

- Позволяет спамить тайджутсу за счёт очень быстрого регена стамины. Улучшенный блок заменяется sky-run.
- Мгновенный раш ломает **обычный** блок оппонента на ближней дистанции.
- После захвата рашем можно сразу использовать любой скилл — очень эффективно.
- Скип хилки работает мгновенно благодаря рашу.
- Подходит только для 1v1.

**Хорош с расами:** Celestial, Puppet, Snake

## Races

Расу можно выбить только при отсутствии красной ДНК на иконке. Повторно выбить нельзя — нужно заново получить ДНК.

### Celestial (Kagoku Event Boss)

- +2000 макс. здоровья (итого 12000).
- Двойной Shift — полёт в любом направлении на 2 сек. со скоростью Shock Cloak. КД: 4 сек.
- Блок поглощает больше дамага.
- Полёт позволяет продолжать дальние комбо (замена Chi Dash).

### Puppet (Ailee Minakami, Dunes)

- Двойной прыжок в любом направлении. Без мода лучше комбинировать с Wide Dash.
- В прыжке доступны все действия: скиллы, хил чакры, блок для движения.
- Активируйте мод с instant dash → Wide Dash вперёд → нажмите блок. Даёт огромную скорость передвижения.
- Иммунитет к эффекту горения от Fire, Inferno и т.д.

### Snake (Kabu Cobra, Obelisk)

- Регенерация здоровья при получении ядовитого урона.
- Кнопка E: выстрел жёлтыми энергетическими змеями с телепортацией. Расход: 30 000 чи. КД: 20 сек.
- Ограничение в составлении сетов.

**Хорош с менторами:** Shindai, Narumaki, Bruce

::: warning Бесполезные расы
**Shinobi / Shark / Human** — не стоят внимания.
:::

## Terms

**Perfect Block (ПБ)** — после получения 200к+ дамага можно блокировать даже в стане. Блок+Атака (ПКМ+ЛКМ) выпускает из комбо с коротким iFrame. После — нельзя использовать M1 в течение 4–5 секунд.

**End-lag** — кратковременный стан в конце скилла. Есть почти у каждого скилла.

**Block Break (ББ)** — мув, ломающий блок.

**iFrame** — скилл, дающий полную неуязвимость на короткое время.

**Grab move** — привязка к противнику во время анимации с нанесением урона.

**Andodge (Antidodge)** — скиллы/механики, блокирующие использование доджа.

**Ragequit (RQ)** — выход из игры во время дангера/PvP.

### Stun Types

- **Placelock Stun** — привязывает модельку и хитбокс к точке. Самый надёжный. Примеры: Bankai 2nd move, Rykan Blade.
- **AOE Stun** — зона, в которой игрок постоянно станится. Примеры: Powder 2nd move, Sound Ken 4th move.
- **Ragdoll Stun** — отключает физику игрока. Ненадёжен. Примеры: Ember Entry, Acrobat Style, Raion-Rengoku 3rd move.
- **Knockback Stun** — стан от отталкивания. Примеры: Dunes Fate Spirit 1st stage, Mecha-Spirit 1st move.

## Slot Bug

::: warning Важно — прочитайте до конца
Чтобы увеличить количество слотов (например, с 2 до 5, с 3 до 7 или с 4 до 8):

1. Зайдите с телефона.
2. Создайте сет — выставьте нужные скиллы заранее и сохраните его.
3. Возьмите блуд, который хотите забагать.
4. Нажмите Load сета и **быстро** перейдите в раздел блудов — нажмите на нужный скилл до того, как он сменится.

**Ограничения:**
- Не работает с элементами.
- Если ставите скиллы на вторую панель (FGH), а не на основную (VBN) — обязательно имейте настоящий блуд на FGH, иначе появится ошибка «Must have equipped Bloodline/Element».
:::
